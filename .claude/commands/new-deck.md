Build an Ableneo presentation outline for: **$ARGUMENTS**

Steps:
1. Use the `brand-strategist` to lock the strategic intent, which category entry point and audience tier this deck serves, and the one thing it must prove.
2. Use the `presentation-builder` to produce a slide-by-slide outline following the
   **Problem → Shift → Our answer → Proof → Future** arc. One idea per slide; numbers over adjectives; mark `[METRIC NEEDED]` where real data is missing.
3. Use the `copywriter` to sharpen every headline (glass-wall test) and key line.
4. Use the `visual-designer` to add color/type notes per slide (Lime + Charcoal + White, one chromatic family, Ableneo type hierarchy, logo/safe-zone on title & closing).
5. Run the whole thing past the `brand-guardian` and report its verdict.

If no audience is given, default to the primary tier (CIO/CTO/COO, measurable ROI).
Honor `ableneo/ABLENEO.md` and `ableneo/knowledge/` throughout. Do not fabricate metrics or brand facts.
