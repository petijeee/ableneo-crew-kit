# Templates

Drop official Ableneo source files here so the crew can build against them directly.

## Expected

- `ableneo_presentation_template.pptx`, **not yet provided.** Once added, the
  `presentation-builder` agent follows its slide masters and layouts. Until then it
  builds from the documented design system (`knowledge/color-palette.md`,
  `knowledge/typography.md`, `knowledge/logo-and-iconography.md`).

When you add the template, update `knowledge/design-tokens.json` with any exact values
it pins down (e.g. grey hex codes that the guidelines deck showed only as swatches).
